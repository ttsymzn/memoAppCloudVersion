# マークダウン機能

メモアプリはマークダウン記法によるリッチテキスト表示に対応しています。以下の記法を使用して、メモをより表現豊かに記述できます。

## 対応している記法

### 見出し
行頭に `#` を付けることで見出しを作成できます。`#` の数でレベルを指定します（1〜6）。
```markdown
# 見出し 1
## 見出し 2
### 見出し 3
```

### リスト
行頭に `-`, `*`, `+` を付けることで箇条書きリストを作成できます。
```markdown
- リストアイテム 1
- リストアイテム 2
```

番号付きリストは数字とドット `1.` を使用します。
```markdown
1. 手順 1
2. 手順 2
```

### 強調（太字・斜体）
- **太字**: `**強調したいテキスト**` または `__強調したいテキスト__`
- *斜体*: `*斜体テキスト*` または `_斜体テキスト_`
- ~~取り消し線~~: `~~削除したテキスト~~`

### 引用
行頭に `>` を付けることで引用ブロックを作成できます。
```markdown
> これは引用です。
> 複数行の引用も可能です。
```

### コードブロック
バッククォート3つ ` ``` ` で囲むことでコードブロックを作成できます。言語を指定するとシンタックスハイライトのようなスタイルが適用されます。

```markdown
\`\`\`javascript
console.log('Hello World');
\`\`\`
```
※ ドキュメント表示の都合上、バッククォートをエスケープしていますが、実際は ` ``` ` と入力してください。

### インラインコード
単語や短いフレーズをバッククォート `` ` `` で囲むことでインラインコードとして表示できます。
例: `constants.js` ファイルを参照してください。

### 水平線
3つ以上のハイフン `---`、アスタリスク `***`、アンダースコア `___` で水平線を作成できます。

```markdown
---
```

## その他（既存機能との統合）

### リンク
URL、ファイルパス、メールアドレスは自動的にリンクに変換されます。
- https://example.com
- file:///path/to/file

### チェックボックス (TODONE機能)
- `[ ]` 未完了タスク
- `[x]` 完了タスク

## 注意事項

- マークダウンのレンダリングは「閲覧モード」でのみ適用されます。編集モードでは生のテキストが表示されます。
- 検索中は、検索語句のハイライトを優先するため、一部のマークダウン記法が無効になる場合があります（リンクやチェックボックスは維持されます）。
